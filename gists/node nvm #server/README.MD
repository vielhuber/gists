### install nvm
- ```curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash```

### install node versions
- ```nvm install node```
- ```nvm install --lts```
- ```nvm install 12.10.0```

## show node versions
- ```nvm ls```
- ```nvm ls-remote```

## switch node versions
- ```nvm use node```
- ```nvm use node --lts```
- ```nvm use 12.10.0```
